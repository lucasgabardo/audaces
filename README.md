<h2 align="center">Install and Run</h2>

Clone the repo:
```bash
$ git clone https://github.com/lucasgabardo/audaces.git
```

Install Angular App dependencies and run:

audaces-app/

```bash
$ cd audaces-app
$ npm install
$ npm start
```
